RSM222 EXAM AID SHEET - COMPREHENSIVE
COST ESTIMATION (High-Low Method)
VC/Unit = (High Cost - Low Cost) ÷ (High Activity - Low Activity)
Use highest & lowest activity levels
Fixed Cost = Total Cost - (VC/Unit × Activity)
Cost Formula: Y = Fixed + (VC/Unit × X)
Y = total cost, X = activity level
CVP ANALYSIS (Ch 3)
CM/Unit = Price - Variable Cost
CM Ratio = CM ÷ Sales = 1 - Variable Cost Ratio
Break-Even Units = Fixed Costs ÷ CM/Unit
Break-Even $ = Fixed Costs ÷ CM Ratio
Target Profit Units = (Fixed + Target) ÷ CM/Unit
Before taxes
Target Profit (after tax) = (Fixed + Target÷(1-tax%)) ÷ CM/Unit
If given after-tax profit
Margin of Safety = Actual - Break-Even
Operating Leverage = Total CM ÷ Operating Income
Higher leverage = more volatile profit
% Change Profit = DOL × % Change Sales
Use for sensitivity analysis
JOB ORDER COSTING (Ch 4-5)
POHR = Estimated Overhead ÷ Estimated Base
Base: DLH, DL$, Machine Hrs, etc.
Applied OH = POHR × Actual Base Used
Job Cost = DM + DL + Applied OH
Overhead Variance = Actual OH - Applied OH
Underapplied (Debit) if Actual > Applied
Overapplied (Credit) if Applied > Actual
Disposition: Close to COGS (if small) or allocate to WIP/FG/COGS
PROCESS COSTING (Ch 6) - Weighted Avg
EU = Physical Units × % Complete
Calculate separately for Materials & Conversion
Materials EU: Check when added!
Beginning: 100%, End: 100% if added at start
End: 0% if added at end
Cost/EUP = (Beg WIP $ + Current $) ÷ Total EU
Weighted avg combines beginning + current
Transferred Out $ = Units Transferred × Cost/EUP
Ending WIP $ = WIP EU × Cost/EUP
Total Costs = Transferred + Ending WIP (reconcile!)
ACTIVITY-BASED COSTING (Ch 7)
Activity Rate = Cost Pool ÷ Total Driver Activity
Calculate rate for each activity
Product Cost = Σ(Activity Rate × Product's Usage)
Sum across all activities
Better accuracy for diverse products
Traditional: one rate, ABC: multiple rates
VAR vs ABSORPTION COSTING (Ch 8) ⭐
Product Costs:
Absorption: DM+DL+Var MOH+Fixed MOH
Variable: DM+DL+Var MOH (Fixed MOH=Period)
Unit Cost (Abs) = Var Cost + (Fixed MOH ÷ Units Produced)
Unit Cost (Var) = Var Cost Only
Income Difference:
Diff = Change in Inv × Fixed MOH/Unit
Prod = SalesSame Income
Prod > Sales (Inv↑)Absorption > Variable
Prod < Sales (Inv↓)Variable > Absorption
Variable: CM format | Absorption: Gross Margin format
BUDGETING (Ch 9)
Production = Sales + Ending Inv - Beginning Inv
Ending of one period = Beginning of next
Mat Purchases = Prod Needs + Ending Mat - Beginning Mat
Prod Needs = Units to Produce × Mat/Unit
Labor Cost = Units to Produce × Hrs/Unit × Rate/Hr
Cash = Beginning + Receipts - Disbursements
Use cash basis (timing matters!)
Flow: Sales → Prod → Mat/Lab/OH → Cash → Statements
VARIANCES (Ch 10)
Materials:
Price Var = AQ × (AP - SP)
Use actual quantity purchased
Quantity Var = SP × (AQ - SQ)
SQ = Units PRODUCED × Std/Unit (not sold!)
Labor:
Rate Var = AH × (AR - SR)
Efficiency Var = SR × (AH - SH)
Variable OH:
Spending = Actual - (AH × VOHR)
Efficiency = VOHR × (AH - SH)
Fixed OH:
Budget = Actual - Budgeted
Volume = Budgeted - Applied
F = Actual < Std, U = Actual > Std
DECISION MAKING (Ch 12)
Relevant = Future + Different + Avoidable
Sunk = Always Irrelevant
Make-or-Buy: Compare Make (Var+Avoid Fixed) vs Buy
Special Order: Accept if Price > Var Costs
If capacity available
Product Mix: Rank by CM per Constrained Resource
Not CM per unit!
Keep/Drop: Compare Segment Margin vs Avoidable Costs
ROI & RESIDUAL INCOME (Ch 13)
ROI = NOI ÷ Avg Operating Assets
ROI = Margin × Turnover
Margin = NOI÷Sales, Turnover = Sales÷Assets
RI = NOI - (Assets × Required ROI)
ROI: %, may discourage investments
RI: $, encourages profitable investments
BALANCED SCORECARD (Ch 11)
1. Financial: ROI, Profit, Growth
2. Customer: Satisfaction, Share, Retention
3. Internal: Quality, Cycle Time, Efficiency
4. Learning: Training, Employee Satisfaction
Chain: Learning → Processes → Customer → Financial
QUICK REFERENCE:
• Variable: Total↑, Per unit constant
• Fixed: Total constant, Per unit↓
• Direct/Indirect ≠ Variable/Fixed
• Product: Inventory→COGS
• Period: Expense immediately
• Relevant: Future+Different
• Sunk: Always ignore
